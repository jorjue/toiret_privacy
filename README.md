# トイレのせせらぎアプリ 🚽💧
静かな環境で安心できる「流水音」アプリです。  
スマホでもPCでも動作し、PWA対応でホーム画面から起動できます。

## 🎵 主な機能
- フェード付き再生・停止
- ループON/OFF切り替え
- 音量スライダー（パーセンテージ表示）
- 前回設定の音量記憶
- 残り再生時間のカウントダウン
- オフライン対応（PWA）

## 🛠️ 技術

- Web Audio API
